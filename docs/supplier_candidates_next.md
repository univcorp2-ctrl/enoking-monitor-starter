| Yahoo!ショッピング | API | 商品検索API v3がJANコード検索とJSON取得に対応。既に実装済み。 |

## 実装ルール

- `parser_hint=search_discovery`: 価格・在庫を買い判定に使わない。
- `parser_hint=manual_check`: 自動取得してもよいが、売買判断は人間確認前提。
- `expected_price_yen` は、確実な個別商品URLで価格/状態/送料が安定して読めるまで空欄にする。
- 個別商品URLが見つかったら `config/supplier_urls.csv` に昇格し、専用パーサーを小さく追加する。
- 連続アクセスを避けるため、Discovery系はGitHub Actionsの通常スケジュールとは別に日次/数日に1回に分けるのが望ましい。
